## 核心理念

### Claude Code 是什么？

Claude Code 不是：^[extracted]

- 代码补全工具
- 聊天机器人加文件读取能力
- 简单的问答工具

Claude Code 是：^[extracted]

- 住在终端里的 AI 编程同事
- 智能体式编程（Agentic Coding）
- 理解目标后自主展开行动的编程智能体

### 开发方式的变化

过去：工具在人后面跟随
- IDE 补全
- Linter 挑错
- CI 验证
- 文档查阅

现在：智能体会主动展开任务^[extracted]
